Contents Platform

Studyall is a platform designed for the administration of learning contents, which can be distributed through packages so the users download only what they want to study.

These packages include texts, questions, tests and tools which can be created from the admin panel to be distributed to the users of the mobile app. It also includes a chatbot that can be trained about any topic.

  • web application , with features like:
    • administration of the subjects / topics that are available for download in the mobile application with function to notify updates;
    • content upload (including questions, texts, tests);
    • interface to train the chatbot in various topics;
    • synchronization with the app through QR code to view any content in a PC browser (Studyall Link );
  • android application, with features like:
    • function to download subjects / topics that are available on the server;
    • notifications when an installed subject is modified on the server, to know when new content is added or something is updated.
    • display of text contents with images;
    • question viewer with functions to mark favorites;
    • exams from the questions loaded on the server for each subject.
    • history of the tests made and report of errors / successes.
    • connection to chat with the server bot to improve learning.
    • integrated QR code reader to connect with Study All Link.
Tech wrap-up
Web Application
  • ASP.NET with C#
  • Serves 2 purposes at once:
    • as API for the mobile app,
    • as administration panel.
  • Connection with Firebase to send push notifications.
  • Chatbot developed with AIML.
  • ASMX Web Services for different functions.
  • SQL Server database.
Mobile Application
  • Native application developed with Android Studio.
  • QR code reader integrated within the application.
  • Function to download files from the server and unzip, for the installation of packages.
  • Integration with Firebase to receive push notifications.
  • Communication with the server through SOAP.
  • Integration with SQLite to store user data.
