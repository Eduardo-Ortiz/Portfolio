Easy and Quick way to order in Restaurants

Ordenus is a system designed to facilitate the ordering process in restaurants. This is achieved through a mobile application that allows customers to see the menu, with photos and details of each dish, and from it they can make their order.

The orders will be displayed in an application for the kitchen, in which all the orders can be displayed and organized in a simple way. This includes different details like from wich table it comes from, the time it was recived, among others.

  • desktop application, with features like:
    • receive orders from customers;
    • calculation of the process time it will take to complete each order;
    • function to maintain control of the orders that have been waiting the longest through alerts with colors and times;
    • ability to see the detail of the orders, with the amount of each dish, the included notes, the table from which it comes and the order number;
  • android application, with features like:
    • configuration to assign the application to a table
    • barcode scanner for bill management;
    • restaurant menu with images, descriptions and prices;
    • shopping cart to add items and generate the order;
    • function to add notes or comments to the order of each product;
    • summary of the total of the bill with detail of everything ordered;
    • deployed notifications to alert when the customer order has been marked as completed in the kitchen;
Tech wrap-up
Desktop Application
  • Java application with Maven
  • Serves 2 purposes at once:
    • as API for the mobile app,
    • as a panel to show the orders in the kitchen.
  • Connection with Firebase to send push notifications.
  • REST services through the Jersey library.
  • MySQL database.
Mobile Application
  • Native application developed with Android Studio
  • Bar code reader integrated within the application.
  • Firebase integration to receive push notifications.
  • Communication with the server through REST using the library Retrofit.
  • Integration of the Picasso library for efficient image management.
  • Integration with SQLite to store user data.
