Control Escolar Mejorado

Testidy es una plataforma que facilita el control escolar, principalmente la asistencia. Para eso se utilizan códigos QR en las aulas que posteriormente son escaneados por los alumnos para confirmar que están dentro de la clase.

Además de esto se incluyen otras funciones adicionales que son útiles para una institución educativa, que incluyen horarios, practicas, avisos, etc. A continuación se listan las principales características del proyecto:

  • aplicación web, con funciones como:
    • accesos separados para alumnos, maestros y administradores;
    • registro de laboratorios con generación automática de códigos QR;
    • función para pase de asistencia durante la hora de clase;
    • generación de reportes de asistencia en tiempo real;
    • horarios con detalles para conocer qué clase se lleva a cabo, bajo qué materia, en qué lugar y con qué maestro;
    • notificaciones push para enviar avisos a alumnos específicos, grupos o a toda la escuela;
    • sistema de exámenes integrado para realizar prácticas durante la hora clase;
    • … y algunas funciones extra como separar aulas, almacenamiento de archivos, importación de alumnos y maestros a través de .csv, entre otras.
  • aplicación móvil para Android, con funciones que incluyen:
    • escaneo de códigos QR para confirmar asistencia a clases;
    • acceso directo a la página web de la escuela;
    • función para solicitar aulas en cualquier fecha y hora disponible;
    • historial para consultar avisos y notificaciones recibidas.
Explicación Técnica
Aplicación Web
  • ASP.NET con C#
  • Sirve para 3 propósitos:
    • como una aplicación web para alumnos y maestros,
    • como API para la aplicación móvil,
    • como panel de administrador.
  • Conexión con Firebase para enviar notificaciones push.
  • Integración con jQuery para widget de selector de fechas y filtros de búsqueda con las funciones de autocompletar.
  • Servicios Web ASMX para distintas funciones.
  • Base de datos en SQL Server.
Aplicación Móvil
  • Aplicación nativa desarrollada con Android Studio
  • Lector de códigos QR integrado dentro de la aplicación.
  • Integración fon Firebase para recibir notificaciones push.
  • Comunicación con el servidor a través de SOAP.
