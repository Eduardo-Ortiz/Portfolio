Ordena en Restaurantes Facil y Rapido

Ordenus es un sistema diseñado para facilitar el proceso de ordenar en los restaurantes. Esto se logra a través de una aplicación móvil que permite a los clientes ver el menú, con fotos y detalles de cada platillo desde el cual pueden hacer su pedido.

Los pedidos serán mostrados en una aplicación destinada a la cocina, en la cual se podrán visualizar y ordenar de una manera sencilla todas las ordenes con distintos detalles como hace cuanto se recibió, de que mesa viene, entre otros.

  • aplicación de escritorio, con funciones como:
    • recibir pedidos de los clientes;
    • calculo del tiempo de proceso que tomara llevar a cabo cada orden;
    • función para mantener un control de las ordenes que llevan más tiempo en espera a través de alertas con colores y tiempos;
    • capacidad para ver el detalle de las ordenes, con la cantidad de cada platillo, las notas incluidas, la mesa de la que viene y el número de orden;
  • aplicación móvil para Android, con funciones que incluyen:
    • configuración para asignar la aplicación a una mesa del restaurante
    • escáner de código de barras para manejo de las cuentas;
    • desplegado del menú del restaurante con imágenes, descripciones y precios;
    • carrito de compra para agregar elementos y generar el pedido;
    • función para agregar notas o comentarios al pedido de cada producto;
    • historial de todos los pedidos realizados durante la orden en cuestión;
    • resumen del total de la cuenta con detalle de todo lo ordenado;
    • desplegado de notificaciones para avisar cuando la orden del cliente ha sido marcada como completada en la cocina;
Explicación Técnica
Aplicación de Escritorio
  • Aplicación Java con Maven
  • Sirve para 2 propósitos:
    • como API para la aplicación móvil,
    • como panel para desplegar ordenes en la cocina.
  • Conexión con Firebase para enviar notificaciones push.
  • Servicios REST a través de la libería Jersey.
  • Base de datos en MySQL.
Aplicación Móvil
  • Aplicación nativa desarrollada con Android Studio
  • Lector de códigos de barra integrado dentro de la aplicación.
  • Integración fon Firebase para recibir notificaciones push.
  • Comunicación con el servidor a través de REST con la librería Retrofit.
  • Integración de la librería Picasso para manejo eficiente de imagenes.
  • Integración con SQL Lite para almacenar datos del usuario.
