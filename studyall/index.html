Plataforma de Contenidos

Studyall es una plataforma diseñada para la administración de contenidos de aprendizaje, los cuales pueden ser distribuidos a través de paquetes para que los usuarios descarguen solo aquello que necesitan.

Estos paquetes incluyen textos, preguntas, exámenes y herramientas los cuales pueden ser creados desde el panel de administrador para ser distribuidos a los usuarios de la app móvil. Además se incluye un chatbot que puede enseñar acerca de diversos contenidos.

  • aplicación web, con funciones como:
    • administración de las materias/tópicos que están disponibles para descargar en la aplicación móvil con función para notificar actualizaciones;
    • carga de contenidos (incluyendo preguntas, textos, exámenes);
    • interface para entrenar al chatbot en diversos temas;
    • sincronización con la app a través de código QR. para visualizar cualquier contenido en un navegador de PC (Studyall Link);
  • aplicación móvil para Android, con funciones que incluyen:
    • función para descargar materias/tópicos que están disponibles en el servidor;
    • notificaciones cuando una materia instalada es actualizada en el servidor, para mantenerse al día los nuevos contenidos agregados o editados.
    • visualizador de contenidos de texto con imágenes;
    • visualizador de preguntas con funciones para marcar favoritos;
    • exámenes a partir de las preguntas cargadas en el servidor para cada materia.
    • historial de los exámenes realizados y reporte de errores/aciertos.
    • conexión con el bot del servidor para mejorar aprendizaje.
    • lector de códigos QR integrado para conectar con Studyall Link.
Explicación Técnica
Aplicación Web
  • ASP.NET con C#
  • Sirve para 2 propósitos:
    • como API para la aplicación móvil,
    • como panel de administrador.
  • Conexión con Firebase para enviar notificaciones push.
  • Chatbot desarrollado con AIML.
  • Servicios Web ASMX para distintas funciones.
  • Base de datos en SQL Server.
Aplicación Móvil
  • Aplicación nativa desarrollada con Android Studio
  • Lector de códigos QR integrado dentro de la aplicación.
  • Función para descargar archivos desde el servidor y descomprimir, para la instalación de los paquetes de materias/tópicos.
  • Integración fon Firebase para recibir notificaciones push.
  • Comunicación con el servidor a través de SOAP.
  • Integración con SQL Lite para almacenar datos del usuario.
