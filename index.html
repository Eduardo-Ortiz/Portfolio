Surveys System

Includes functions like different types of questions, audio recording for verifications, capture of locations and integration with Google Maps to assign work areas and pollsters monitoring. Contains reporting tools to make custom graphs and even an user-friendly query generator to analyze obtained results.

Used Technology

Laravel was used in the backend and Vue.js in the frontend. The mobile application was developed using Android Studio (Java). The communication between both sides was done with a REST API. For the integration with maps, the Google Maps API was used, and at last Firebase was included for the notifications.

Intelligent Study

Heugen is a Web application designed to make easier the task of memorizing contents or concepts. To achieve this, an evaluation and repetition technique is used with an algorithm that analyzes and gives priority to the contents that should be studied more, based on the results obtained by the user.

Used Technology

The Laravel framework was used in the backend and Vue.js in the frontend. The decision algorithms were written using PHP. As for the database, MySQL is used.

Improved School Control

Platform designed for the Faculty of Mechanical and Electrical Engineering North Unit of Monclova, Coahuila Mexico. It integrates a web system and a mobile application to control the asistance of students through QR codes, in addition to managing schedules, tests and other aspects.

Used Technology

ASP.NET was used with C # in the backend and jQuery in the frontend. The mobile application was created with Android Studio (Java). The communication between both sides was done through Web Services with SOAP. SQL Server was used for the database.
